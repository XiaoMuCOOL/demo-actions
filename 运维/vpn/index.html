shadowsocks、outline 的搭建。

# 删除阿里云云盾、安骑士等安全组件

不删除可能有被封 IP 风险，所以自己百度下如何删除吧，可参考此文章 (opens new window)。

# 安装docker

// 更新
sudo apt update
// Ubuntu 自带docker.io源
sudo apt install docker.io -y

# 下载shadowsocks镜像

// 下载镜像
sudo docker pull mritd/shadowsocks
// 启动镜像
sudo docker run -dt --name ss -p [端口]:6443 mritd/shadowsocks -s "-s 0.0.0.0 -p 6443 -m aes-256-cfb -k [密码] –fast-open"

成功！

然后下载客户端配置下设置，就能愉快的访问P站学习了。

客户端下载：windows (opens new window)、android (opens new window)、mac (opens new window) IOS 暂无官方客服端，如果想要支持 iPhone，那就搭建 Outline 吧，但 Outline 协议听说比较老，容易被封。

# 搭建 Outline 服务器

安装好 Docker 后，在服务器输入命令：

sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/Jigsaw-Code/outline-server/master/src/server_manager/install_scripts/install_server.sh)"

然后设置防火墙，允许端口 TCP 18339, TCP and UDP 47902 通过。

下载 Outline Manager (opens new window)并安装。

打开 Outline Manager 添加服务器，选择最后一项随时随地安装 Outline，在第二步输入框内填入刚刚服务器生成的 apiUrl 开头的 JSON 字符串。我的为：

{"apiUrl":"[IP]","certSha256":"[密钥]"}

点击完成即添加成功服务器。

然后点击添加新密钥，输入名称，点击分享，复制生成的 ss:// 开头链接给客户端使用即可链接！

最后，下载 Outline Client (opens new window) 应用程序。到目前为止，它支持 macOS，Windows，Linux，ChromeOS，iOS 和 Android。

在应用程序中，点击加号按钮，粘贴密钥，然后连接到服务器。

方便！完美！
