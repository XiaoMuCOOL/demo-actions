VScode搭建JAVA开发环境

发布时间 : 2019-01-01 最后修改 : 2022-05-30
下一篇 : Mu教程系列：VPN 搭建

自从转到前端后，已经好久没弄过JAVA了，VScode非常强大，那就用它来搭建一套JAVA的开发环境吧。

# 安装

# JDK

官网下载jdk (opens new window)，然后一步步安装就行，我安装目录为：C:/Program Files/Java/jdk1.8.0_191。

# Mevan

maven (opens new window)官网下载JAR包，解压至JAVA安装目录，我的在C:/Program Files/Java/apache-maven-3.6.3。

打开apache-maven-3.6.3/conf/settings.xml，添加阿里云仓库，默认仓库比较慢：

<!-- 阿里云仓库 -->
<mirror>
  <id>alimaven</id>
  <mirrorOf>central</mirrorOf>
  <name>aliyun maven</name>
  <url>http://maven.aliyun.com/nexus/content/repositories/central/</url>
</mirror>
<mirror>
  <id>nexus-aliyun</id>
  <mirrorOf>*</mirrorOf>
  <name>Nexus aliyun</name>
  <url>http://maven.aliyun.com/nexus/content/groups/public</url>
</mirror>

# VScode

在VScode的Extensions(Ctrl+Shift+X)菜单下搜索Java Extension Pack和Spring Boot Extension Pack插件开发包，并安装。

然后在文件->首选项->设置里点击在setting.json中编辑：

"java.home": "C:\\Program Files\\Java\\jdk1.8.0_191",
"maven.terminal.useJavaHome": true,
"maven.executable.path": "C:\\Program Files\\Java\\apache-maven-3.6.3\\bin\\mvn",
"java.configuration.maven.userSettings": "C:\\Program Files\\Java\\apache-maven-3.6.3\\conf\\settings.xml",
"maven.terminal.customEnv": [
  {
    "environmentVariable": "JAVA_HOME",
    "value": "C:\\Program Files\\Java\\jdk1.8.0_191"
  }
]

环境基本配置完成了，最重要的是实战了。
